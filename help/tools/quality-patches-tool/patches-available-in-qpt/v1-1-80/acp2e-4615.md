# ACP2E-4615: Falha nos reembolsos de pedido online do PayPal com o erro *O gateway do PayPal rejeita a solicitação. Erro Interno.*

O patch ACP2E-4615 corrige o problema em que os reembolsos de pedidos online falham com um erro do PayPal informando: *O gateway do PayPal rejeita a solicitação. Erro Interno.*. Este patch está disponível quando o [[!DNL Quality Patches Tool (QPT)]](/help/tools/quality-patches-tool/quality-patches-tool-to-self-serve-quality-patches.md) 1.1.80 está instalado. A ID do patch é ACP2E-4615. Observe que esse problema está programado para ser corrigido no Adobe Commerce 2.4.9.

## Produtos e versões afetados

**O patch foi criado para a versão do Adobe Commerce:**

* Adobe Commerce (todos os métodos de implantação) 2.4.7-p8

**Compatível com as versões do Adobe Commerce:**

* Adobe Commerce (todos os métodos de implantação) 2.4.5 - 2.4.9

>[!NOTE]
>
>O patch pode se tornar aplicável a outras versões com as novas versões do [!DNL Quality Patches Tool]. Para verificar se o patch é compatível com a sua versão do Adobe Commerce, atualize o pacote `magento/quality-patches` para a versão mais recente e verifique a compatibilidade na [[!DNL Quality Patches Tool]: página Procurar patches](https://experienceleague.adobe.com/tools/commerce-quality-patches/index.html). Use a ID do patch como palavra-chave de pesquisa para localizar o patch.

## Problema

Falha nos reembolsos de pedidos online com um erro do PayPal informando: *O gateway do PayPal rejeita a solicitação. Erro Interno.*.

<u>Etapas a serem reproduzidas</u>:

1. Crie um produto com um `price` = *$100*.
1. Faça o pedido usando *$50* de crédito da loja e *PayPal* como método de pagamento.
1. Criar uma fatura para o pedido.
1. Comece a criar um memorando de crédito com `Adjustment Fee` = *$50*, para que o `Grand Total` = *$0*.
1. Clique em **[!UICONTROL Refund]** para executar um reembolso online.

<u>Resultados esperados</u>:

Não deve ser possível criar reembolsos com um `Grand Total` = *$0*. O botão **[!UICONTROL Refund]** deve ser desabilitado.

<u>Resultados reais</u>:

1. O memorando de crédito não foi criado. O erro *Gateway do PayPal rejeita a solicitação. Erro Interno (#10001: Erro Interno).* é exibida.
