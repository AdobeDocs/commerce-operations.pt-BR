# ACP2E-4533: Falha ao carregar imagens de espaço reservado na loja quando o URL inclui código de loja

O patch ACP2E-4533 corrige o problema em que imagens de espaço reservado não são carregadas na loja quando um código de armazenamento é incluído no URL. Este patch está disponível quando o [[!DNL Quality Patches Tool (QPT)]](/help/tools/quality-patches-tool/quality-patches-tool-to-self-serve-quality-patches.md) 1.1.80 está instalado. A ID do patch é ACP2E-4533. Observe que esse problema está programado para ser corrigido no Adobe Commerce 2.4.9.

## Produtos e versões afetados

**O patch foi criado para a versão do Adobe Commerce:**

* Adobe Commerce (todos os métodos de implantação) 2.4.7-p8

**Compatível com as versões do Adobe Commerce:**

* Adobe Commerce (todos os métodos de implantação) 2.4.7 - 2.4.7-p10

>[!NOTE]
>
>O patch pode se tornar aplicável a outras versões com as novas versões do [!DNL Quality Patches Tool]. Para verificar se o patch é compatível com a sua versão do Adobe Commerce, atualize o pacote `magento/quality-patches` para a versão mais recente e verifique a compatibilidade na [[!DNL Quality Patches Tool]: página Procurar patches](https://experienceleague.adobe.com/tools/commerce-quality-patches/index.html). Use a ID do patch como palavra-chave de pesquisa para localizar o patch.

## Problema

As imagens do espaço reservado não são carregadas na loja quando o URL inclui o código da loja.


<u>Etapas a serem reproduzidas</u>:

1. Crie um site adicional, loja e visualização de loja.
1. Para configurar imagens de espaço reservado no escopo padrão, na barra lateral Admin, vá para **[!UICONTROL Stores]** > *[!UICONTROL Settings]* > **[!UICONTROL Configuration]**.
1. No painel esquerdo, expanda **[!UICONTROL Catalog]** e escolha **[!UICONTROL Catalog]**.
1. Expanda a seção **[!UICONTROL Product Image Placeholders]** para configurar imagens de espaço reservado para o novo escopo de site.
1. Crie um produto com imagens e atribua-o a ambos os sites.
1. Exclua as imagens do produto carregadas da pasta `pub/media/catalog/product`.
1. Abra o produto na loja de ambos os sites.

<u>Resultados esperados</u>:

O site que não for padrão deve exibir as imagens de espaço reservado específicas do site quando as imagens reais não estiverem disponíveis.

<u>Resultados reais</u>:

O site que não é padrão exibe a imagem de espaço reservado padrão, independentemente das imagens de espaço reservado carregadas para ele.
