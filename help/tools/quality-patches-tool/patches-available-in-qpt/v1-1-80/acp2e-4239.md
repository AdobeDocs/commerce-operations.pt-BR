# ACP2E-4239: Os filtros de Data da Grade de Admin não retornam resultados devido à incompatibilidade de Fuso Horário

O patch ACP2E-4239 corrige o problema em que os filtros de grade de administração que usam atributos de data não retornam resultados devido à incompatibilidade de fuso horário entre a data selecionada, os valores [!DNL UTC] armazenados e o fuso horário de armazenamento configurado. Este patch está disponível quando o [[!DNL Quality Patches Tool (QPT)]](/help/tools/quality-patches-tool/quality-patches-tool-to-self-serve-quality-patches.md) 1.1.80 está instalado. A ID do patch é ACP2E-4239. Observe que esse problema está programado para ser corrigido no Adobe Commerce 2.4.9.

## Produtos e versões afetados

**O patch foi criado para a versão do Adobe Commerce:**

* Adobe Commerce (todos os métodos de implantação) 2.4.7-p8, 2.4.8-p2

**Compatível com as versões do Adobe Commerce:**

* Adobe Commerce (todos os métodos de implantação) 2.4.7 - 2.4.8-p5

>[!NOTE]
>
>O patch pode se tornar aplicável a outras versões com as novas versões do [!DNL Quality Patches Tool]. Para verificar se o patch é compatível com a sua versão do Adobe Commerce, atualize o pacote `magento/quality-patches` para a versão mais recente e verifique a compatibilidade na [[!DNL Quality Patches Tool]: página Procurar patches](https://experienceleague.adobe.com/tools/commerce-quality-patches/index.html). Use a ID do patch como palavra-chave de pesquisa para localizar o patch.

## Problema

Os filtros de grade de administração que usam atributos de data não retornam resultados devido a uma incompatibilidade de fuso horário entre a data selecionada, os valores [!DNL UTC] armazenados e o fuso horário de armazenamento configurado.

<u>Etapas a serem reproduzidas</u>:

1. Fazer um pedido.
1. Criar uma fatura para o pedido.
1. Crie um memorando de crédito.
1. Vá para **[!UICONTROL Admin]** > **[!UICONTROL Sales]** > **[!UICONTROL Credit Memos]**.
1. Clique em **[!UICONTROL Filter]** e selecione a data atual nos campos **[!UICONTROL Created from]** e **[!UICONTROL Created to]**.
1. Clique em **[!UICONTROL Filter]**.

<u>Resultados esperados</u>:

Os pedidos criados na data selecionada aparecem na grade.

<u>Resultados reais</u>:

A grade não exibe resultados.
