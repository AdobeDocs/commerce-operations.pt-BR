# ACSD-67171: sessão [!DNL B2B] expirada durante o check-out de redirecionamentos para a página Acesso Negado em vez de logon

O patch ACSD-67171 corrige um problema em que [!DNL B2B] clientes são redirecionados para uma página *Acesso negado* quando a sessão expira ou é removida durante o check-out. Este patch está disponível quando o [[!DNL Quality Patches Tool (QPT)]](/help/tools/quality-patches-tool/quality-patches-tool-to-self-serve-quality-patches.md) 1.1.73 está instalado. A ID do patch é ACSD-67171. Observe que esse problema está programado para ser corrigido no Adobe Commerce 2.4.9.

## Produtos e versões afetados

**O patch foi criado para a versão do Adobe Commerce:**

* Adobe Commerce (todos os métodos de implantação) 2.4.8-p1

**Compatível com as versões do Adobe Commerce:**

* Adobe Commerce (todos os métodos de implantação) 2.4.4 - 2.4.8-p3

>[!NOTE]
>
>O patch pode se tornar aplicável a outras versões com as novas versões do [!DNL Quality Patches Tool]. Para verificar se o patch é compatível com a sua versão do Adobe Commerce, atualize o pacote `magento/quality-patches` para a versão mais recente e verifique a compatibilidade na [[!DNL Quality Patches Tool]: página Procurar patches](https://experienceleague.adobe.com/tools/commerce-quality-patches/index.html?lang=pt-BR). Use a ID do patch como palavra-chave de pesquisa para localizar o patch.

## Problema

O sistema mostra uma página de Acesso Negado durante o check-out quando a sessão de um usuário do [!DNL B2B] expira ou é removida, em vez de redirecioná-lo para a página de logon do cliente.

<u>Etapas a serem reproduzidas</u>:

1. No painel Admin, vá para [!UICONTROL Stores] > [!UICONTROL Configuration] > [!UICONTROL General] > [!UICONTROL [!DNL B2B] Features], defina [!UICONTROL [!DNL B2B] Features] como *Sim*, vá para [!UICONTROL Customers] > [!UICONTROL Persistent Shopping Cart] e defina [!UICONTROL Enable Persistence] como *Sim*.
1. Vá para [!UICONTROL Customers] > [!UICONTROL Companies] e selecione [!UICONTROL Add New Company] para criar uma nova Empresa [!DNL B2B].
1. Faça logon na loja como o usuário da Empresa [!DNL B2B].
1. Adicione qualquer produto ao carrinho.
1. Vá para a página Check-out.
1. Permita que a sessão expire.
1. Recarregue a página Check-out.

<u>Resultados esperados</u>:

Você é redirecionado para a página de logon do cliente.

<u>Resultados reais</u>:

Você é redirecionado para `/company/accessdenied` e verá *Você não tem permissão para exibir esta página. Se você acha que isso é um erro, contate o administrador da sua empresa.*
