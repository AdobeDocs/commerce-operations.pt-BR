# ACSD-66807: a tabela `report_viewed_product_index` mostra uma contagem incorreta de exibições de página de produto

O patch ACSD-66807 corrige o problema em que a tabela `report_viewed_product_index` mostra uma contagem incorreta de exibições de página de produto. Este patch está disponível quando o [[!DNL Quality Patches Tool (QPT)]](/help/tools/quality-patches-tool/quality-patches-tool-to-self-serve-quality-patches.md) 1.1.72 está instalado. A ID do patch é ACSD-66807. Observe que esse problema está programado para ser corrigido no Adobe Commerce 2.4.9.

## Produtos e versões afetados

**O patch foi criado para a versão do Adobe Commerce:**

* Adobe Commerce (todos os métodos de implantação) 2.4.7-p5

**Compatível com as versões do Adobe Commerce:**

* Adobe Commerce (todos os métodos de implantação) 2.4.4 - 2.4.8-p2

>[!NOTE]
>
>O patch pode se tornar aplicável a outras versões com as novas versões do [!DNL Quality Patches Tool]. Para verificar se o patch é compatível com a sua versão do Adobe Commerce, atualize o pacote `magento/quality-patches` para a versão mais recente e verifique a compatibilidade na [[!DNL Quality Patches Tool]: página Procurar patches](https://experienceleague.adobe.com/tools/commerce-quality-patches/index.html?lang=pt-BR). Use a ID do patch como palavra-chave de pesquisa para localizar o patch.

## Problema

A tabela `report_viewed_product_index` mostra uma contagem incorreta de exibições de página de produto.

<u>Etapas a serem reproduzidas</u>:

1. Vá para **[!UICONTROL Stores]** > *[!UICONTROL Settings]* > **[!UICONTROL Configuration]** > **[!UICONTROL General]** > **[!UICONTROL Reports]**. Defina **[!UICONTROL Enable "Product View" Report]** como *Sim*.
1. Limpar cache.
1. Visite uma página de produto no front-end (por exemplo, produto &quot;A&quot;).
1. Verifique a tabela `report_viewed_product_index` e veja um registro do produto &quot;A&quot;.
1. Atualize a página (ou abra-a em uma guia incógnita).

<u>Resultados esperados</u>:

Deve haver dois registros na tabela `report_viewed_product_index` para o produto &quot;A&quot;.

<u>Resultados reais</u>:

Existe apenas um registro na tabela `report_viewed_product_index` para o produto &quot;A&quot;.
