# ACSD-69311: Cálculo de imposto incorreto em avisos de crédito após reembolsos parciais de NFFs

O patch ACSD-69311 corrige o problema em que, com cálculos de imposto incorretos em avisos de crédito ao criar uma restituição parcial de uma NFF, se um aviso de crédito anterior tiver sido criado a partir da página de exibição de ordem. Este patch está disponível quando o [[!DNL Quality Patches Tool (QPT)]](/help/tools/quality-patches-tool/quality-patches-tool-to-self-serve-quality-patches.md) 1.1.77 está instalado. A ID do patch é ACSD-69311. Observe que esse problema está programado para ser corrigido no Adobe Commerce 2.4.9.

## Produtos e versões afetados

**O patch foi criado para a versão do Adobe Commerce:**

* Adobe Commerce (todos os métodos de implantação) 2.4.7-p8

**Compatível com as versões do Adobe Commerce:**

* Adobe Commerce (todos os métodos de implantação) 2.4.4 - 2.4.8-p3

>[!NOTE]
>
>O patch pode se tornar aplicável a outras versões com as novas versões do [!DNL Quality Patches Tool]. Para verificar se o patch é compatível com a sua versão do Adobe Commerce, atualize o pacote `magento/quality-patches` para a versão mais recente e verifique a compatibilidade na [[!DNL Quality Patches Tool]: página Procurar patches](https://experienceleague.adobe.com/tools/commerce-quality-patches/index.html). Use a ID do patch como palavra-chave de pesquisa para localizar o patch.

## Problema

Cálculos de imposto incorretos ocorrerão em avisos de crédito ao criar uma restituição parcial a partir de uma NFF, se um aviso de crédito anterior tiver sido criado a partir da página de exibição da ordem.

<u>Etapas a serem reproduzidas</u>:

1. Configure os preços de catálogo para incluir impostos.
1. Configure todos os preços de exibição para incluir impostos.
1. Crie uma alíquota e uma regra de imposto para *10%* para *Austrália*.
1. Crie 2 produtos com os preços *100* e *200*.
1. Adicione ambos os produtos ao carrinho.
1. Confira usando frete grátis e um endereço australiano.
1. Abra o pedido no [!UICONTROL Admin].
1. Criar uma fatura para o pedido.
1. Crie um aviso de crédito parcial usando o botão na página Exibição de pedido selecionando um produto.
1. Abra a fatura.
1. Criar um aviso de crédito usando o botão na fatura.

<u>Resultados esperados</u>:

O valor do imposto é refletido corretamente no aviso de crédito para o reembolso.

<u>Resultados reais</u>:

O segundo aviso de crédito inclui o valor total do imposto no total para o reembolso.
