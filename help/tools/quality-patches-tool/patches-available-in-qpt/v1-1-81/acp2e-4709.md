# ACP2E-4709: As páginas do CMS não são salvas no Page Builder quando a hierarquia da página inclui aspas duplas

O patch ACP2E-4709 corrige o problema em que as páginas do CMS não são salvas no Page Builder quando a hierarquia da página inclui aspas duplas. Este patch está disponível quando o [[!DNL Quality Patches Tool (QPT)]](/help/tools/quality-patches-tool/quality-patches-tool-to-self-serve-quality-patches.md) 1.1.81 está instalado. A ID do patch é ACP2E-4709. Observe que esse problema está programado para ser corrigido no Adobe Commerce 2.4.9.

## Produtos e versões afetados

**O patch foi criado para a versão do Adobe Commerce:**

* Adobe Commerce (todos os métodos de implantação) 2.4.7-p9

**Compatível com as versões do Adobe Commerce:**

* Adobe Commerce (todos os métodos de implantação) 2.4.7 - 2.4.8-p5

>[!NOTE]
>
>O patch pode se tornar aplicável a outras versões com as novas versões do [!DNL Quality Patches Tool]. Para verificar se o patch é compatível com a sua versão do Adobe Commerce, atualize o pacote `magento/quality-patches` para a versão mais recente e verifique a compatibilidade na [[!DNL Quality Patches Tool]: página Procurar patches](https://experienceleague.adobe.com/tools/commerce-quality-patches/index.html). Use a ID do patch como palavra-chave de pesquisa para localizar o patch.

## Problema

As páginas do CMS não são salvas no Page Builder quando a hierarquia da página inclui aspas duplas.

<u>Pré-requisitos</u>:

O Page Builder está ativado.

<u>Etapas a serem reproduzidas</u>:

1. Na barra lateral Admin, vá para **[!UICONTROL Stores]** > **Configurações** > **Configuração**. No painel esquerdo, em **[!UICONTROL General]**, escolha **Gerenciamento de conteúdo**.
1. Expanda **[!UICONTROL CMS Page Hierarchy]**, defina **[!UICONTROL Enable Hierarchy Functionality]** como **[!UICONTROL Yes]** e clique em **[!UICONTROL Save Config]**.
1. Na barra lateral Admin, vá para **Conteúdo** > **[!UICONTROL Elements]** > **[!UICONTROL Pages]** e clique em **[!UICONTROL Add New Page]**.
1. Insira um título de página que inclua aspas duplas, coloque a página sob a raiz do site na guia **Hierarquia** e clique em **[!UICONTROL Save]**.
1. Use o Page Builder para adicionar conteúdo à página e clique em **[!UICONTROL Save]**.

<u>Resultados esperados</u>:

A nova página do CMS é salva com sucesso sem erros.

<u>Resultados reais</u>:

- A nova página do CMS não é salva.
- O console do navegador exibe o seguinte erro: `Uncaught SyntaxError: JSON.parse: expected ',' or '}' after property value in object at line 1 column 79 of the JSON data`.
