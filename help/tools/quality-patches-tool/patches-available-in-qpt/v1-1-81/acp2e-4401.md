# ACP2E-4401: [!UICONTROL Scheduled Update Preview] da Página Inicial com redirecionamentos de produtos configuráveis para a Página de Manutenção

O patch ACP2E-4401 corrige o problema em que o **[!UICONTROL Scheduled Update Preview]** da [!UICONTROL Home Page] com produtos configuráveis redireciona para a **Página de Manutenção** ou mostra um erro em vez de exibir a lista de produtos. Este patch está disponível quando o [[!DNL Quality Patches Tool (QPT)]](/help/tools/quality-patches-tool/quality-patches-tool-to-self-serve-quality-patches.md) 1.1.81 está instalado. A ID do patch é ACP2E-4401. Observe que esse problema está programado para ser corrigido no Adobe Commerce 2.4.9.

## Produtos e versões afetados

**O patch foi criado para a versão do Adobe Commerce:**

* Adobe Commerce (todos os métodos de implantação) 2.4.8-p3

**Compatível com as versões do Adobe Commerce:**

* Adobe Commerce (todos os métodos de implantação) 2.4.8 - 2.4.8-p5

>[!NOTE]
>
>O patch pode se tornar aplicável a outras versões com as novas versões do [!DNL Quality Patches Tool]. Para verificar se o patch é compatível com a sua versão do Adobe Commerce, atualize o pacote `magento/quality-patches` para a versão mais recente e verifique a compatibilidade na [[!DNL Quality Patches Tool]: página Procurar patches](https://experienceleague.adobe.com/tools/commerce-quality-patches/index.html?lang=pt-BR). Use a ID do patch como palavra-chave de pesquisa para localizar o patch.

## Problema

O **[!UICONTROL Scheduled Update Preview]** da [!UICONTROL Home Page] com produtos configuráveis redireciona para a **Página de Manutenção** ou mostra um erro em vez de exibir a lista de produtos.

<u>Etapas a serem reproduzidas</u>:

1. Crie um produto configurável com várias opções e atribua-o a uma categoria.
1. Na barra lateral Admin, vá para **Conteúdo** > **[!UICONTROL Elements]** > **[!UICONTROL Pages]** e abra a página do CMS atribuída como a **Página Inicial**.
1. Na caixa **[!UICONTROL Scheduled Changes]**, clique em **[!UICONTROL Schedule New Update]**, crie uma atualização agendada para a página e salve a atualização.
1. Na caixa **[!UICONTROL Scheduled Changes]**, clique em **[!UICONTROL Preview]**.
1. Na janela de pré-visualização, clique em um link de categoria.

<u>Resultados esperados</u>:

1. A visualização de preparo é exibida corretamente sem erros.
1. O link de categoria abre a página da lista de produtos conforme esperado.

<u>Resultados reais</u>:

1. A navegação de visualização abre a **Página de manutenção** ou dispara um erro fatal.
1. O arquivo `var/log/exception.log` contém um erro semelhante a *`report.CRITICAL: TypeError: Magento\Weee\Plugin\ConfigurableProduct\Pricing\FinalPriceResolver::afterResolvePrice(): Return value must be of type float, string returned in /var/app/code/Magento/Weee/Plugin/ConfigurableProduct/Pricing/FinalPriceResolver.php:51`*.
