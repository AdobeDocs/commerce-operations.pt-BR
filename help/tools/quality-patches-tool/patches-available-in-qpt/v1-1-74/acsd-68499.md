# ACSD-68499: A mutação updateCartItems do GraphQL retorna *sucesso* para quantidades além do estoque disponível

O patch ACSD-68499 corrige o problema em que a mutação `updateCartItems` do GraphQL retorna uma resposta de *sucesso* incorreta ao atualizar quantidades que excedem o estoque disponível, resultando em quantidades e totais inflados. Este patch está disponível quando o [[!DNL Quality Patches Tool (QPT)]](/help/tools/quality-patches-tool/quality-patches-tool-to-self-serve-quality-patches.md) 1.1.74 está instalado. A ID do patch é ACSD-68499. Observe que esse problema está programado para ser corrigido no Adobe Commerce 2.4.9.

## Produtos e versões afetados

**O patch foi criado para a versão do Adobe Commerce:**

* Adobe Commerce (todos os métodos de implantação) 2.4.8-p2

**Compatível com as versões do Adobe Commerce:**

* Adobe Commerce (todos os métodos de implantação) 2.4.4 - 2.4.8-p3

>[!NOTE]
>
>O patch pode se tornar aplicável a outras versões com as novas versões do [!DNL Quality Patches Tool]. Para verificar se o patch é compatível com a sua versão do Adobe Commerce, atualize o pacote `magento/quality-patches` para a versão mais recente e verifique a compatibilidade na [[!DNL Quality Patches Tool]: página Procurar patches](https://experienceleague.adobe.com/tools/commerce-quality-patches/index.html?lang=pt-BR). Use a ID do patch como palavra-chave de pesquisa para localizar o patch.

## Problema

A mutação `updateCartItems` do GraphQL no Adobe Commerce retorna incorretamente a resposta de *sucesso* ao atualizar quantidades além do estoque disponível, causando totais e quantidades infladas do carrinho.

<u>Etapas a serem reproduzidas</u>:

1. Crie um carrinho vazio.
1. Adicione um produto com estoque limitado (por exemplo, 10 unidades disponíveis) ao carrinho.
1. Consulte o carrinho e observe o `cart_item_id`.
1. Atualize a quantidade do item do carrinho para 100 (excede o estoque disponível de 10) usando a seguinte mutação:

   ```
   mutation ($cart: String!, $cartItemId: Int!, $quantity: Float!) {
     updateCartItems(
       input: {
         cart_id: $cart
         cart_items: [{ cart_item_id: $cartItemId, quantity: $quantity }]
       }
     ) {
       cart {
         items {
           product {
             name
           }
           quantity
         }
         prices {
           grand_total {
             value
             currency
           }
         }
       }
     }
   }
   ```

1. Consulte o carrinho novamente para verificar a quantidade real.

<u>Resultados esperados</u>:

A validação de estoque impede atualizações de quantidade inválidas.

<u>Resultados reais</u>:

A mutação `updateCartItems` retorna uma resposta de *sucesso* com uma quantidade de 100.
