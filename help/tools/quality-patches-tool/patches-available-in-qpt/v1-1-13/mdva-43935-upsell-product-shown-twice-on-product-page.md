# MDVA-43935: produto de venda adicional mostrado duas vezes na página do produto

O patch MDVA-43935 corrige o problema em que o produto de venda adicional é mostrado duas vezes na página do produto. Este patch está disponível quando a [Ferramenta de Patches de Qualidade (QPT)](https://experienceleague.adobe.com/pt-br/docs/commerce-operations/tools/quality-patches-tool/quality-patches-tool-to-self-serve-quality-patches) 1.1.13 está instalada. A ID do patch é MDVA-43935. Observe que o problema está programado para ser corrigido no Adobe Commerce 2.4.5.

## Produtos e versões afetados

**O patch foi criado para a versão do Adobe Commerce:**

* Adobe Commerce (todos os métodos de implantação) 2.4.3-p1

**Compatível com as versões do Adobe Commerce:**

* Adobe Commerce (todos os métodos de implantação) 2.4.1 - 2.4.4

>[!NOTE]
>
>O patch pode se tornar aplicável a outras versões com as novas versões da Ferramenta de patches de qualidade. Para verificar se o patch é compatível com a sua versão do Adobe Commerce, atualize o pacote `magento/quality-patches` para a versão mais recente e verifique a compatibilidade na [[!DNL Quality Patches Tool]: página Procurar patches](https://experienceleague.adobe.com/pt-br/docs/commerce-operations/tools/quality-patches-tool/quality-patches-tool-to-self-serve-quality-patches). Use a ID do patch como palavra-chave de pesquisa para localizar o patch.

## Problema

O produto de venda adicional é mostrado duas vezes na página do produto.

<u>Etapas a serem reproduzidas</u>:

1. Criar uma subcategoria.
1. Crie um produto simples.
1. Crie um pacote que use o produto simples como uma opção.
1. Limpar caches e executar reindexação.
1. Abra o produto simples na loja.

<u>Resultados esperados</u>:

O produto do pacote é listado como venda adicional apenas uma vez.

<u>Resultados reais</u>:

O produto do pacote é listado duas vezes.
