# ACP2E-4431: [!UICONTROL Related Products] correspondidos pelas regras de destino são excluídos durante o processo de reindexação

O patch ACP2E-4431 corrige o problema em que o **[!UICONTROL Related Products]** correspondente às regras de destino são excluídos durante o processo de reindexação. Este patch está disponível quando o [[!DNL Quality Patches Tool (QPT)]](/help/tools/quality-patches-tool/quality-patches-tool-to-self-serve-quality-patches.md) 1.1.78 está instalado. A ID do patch é ACP2E-4431. Observe que esse problema está programado para ser corrigido no Adobe Commerce 2.4.9.

## Produtos e versões afetados

**O patch foi criado para a versão do Adobe Commerce:**

* Adobe Commerce (todos os métodos de implantação) 2.4.7-p4

**Compatível com as versões do Adobe Commerce:**

* Adobe Commerce (todos os métodos de implantação) 2.4.4 - 2.4.8-p4

>[!NOTE]
>
>O patch pode se tornar aplicável a outras versões com as novas versões do [!DNL Quality Patches Tool]. Para verificar se o patch é compatível com a sua versão do Adobe Commerce, atualize o pacote `magento/quality-patches` para a versão mais recente e verifique a compatibilidade na [[!DNL Quality Patches Tool]: página Procurar patches](https://experienceleague.adobe.com/tools/commerce-quality-patches/index.html). Use a ID do patch como palavra-chave de pesquisa para localizar o patch.

## Problema

Os **[!UICONTROL Related Products]** correspondentes às regras de destino são excluídos durante o processo de reindexação.

<u>Etapas a serem reproduzidas</u>:

1. Crie 10000 ou mais produtos.
1. Atribua 10000 ou mais produtos a 5000 ou mais categorias.
1. Crie 100 ou mais regras de produto relacionadas.
1. Configure as condições da regra em que **[!UICONTROL Products]** para corresponder ao uso de **[!UICONTROL Product Category]** é *X*.
1. A configuração de **[!UICONTROL Products]** para exibição usando **[!UICONTROL Product Category]** é **[!UICONTROL Constant Value]** *Y*.
1. Execute esta consulta GraphQL para recuperar `related_products` de um produto:

   ```graphql
   {
       products(filter: { url_key: { eq: "simple-product-491" } }) {
           items {
               sku
               related_products {
                   uid
                   sku
                   name
                   url_key
                   stock_status
                   price_range {
                       minimum_price {
                           regular_price {
                              value
                           }
                           final_price {
                               value
                           }
                       }
                   }
               }
           }
       }
   }
   ```

1. Criar uma grande lista de pendências para o índice `targetrule_product_rule`.
1. Criar uma grande lista de pendências para o índice `targetrule_rule_product`.
1. Execute o indexador `targetrule_rule_product`.
1. Execute o indexador `targetrule_product_rule`.

<u>Resultados esperados</u>:

A lista `related_products` é sempre preenchida com produtos que correspondem às condições da regra.

<u>Resultados reais</u>:

A lista `related_products` retorna uma matriz vazia enquanto os indexadores estão em execução.
