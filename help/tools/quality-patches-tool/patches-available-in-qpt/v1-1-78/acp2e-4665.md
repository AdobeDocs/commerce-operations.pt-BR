# ACP2E-4665: Os produtos secundários de produtos configuráveis com vídeos nas galerias do produto não aparecem quando solicitados por meio da API REST

O patch ACP2E-4665 corrige o problema em que produtos secundários de um produto configurável não são retornados pela API REST quando um vídeo é incluído na galeria de produtos. Este patch está disponível quando o [[!DNL Quality Patches Tool (QPT)]](/help/tools/quality-patches-tool/quality-patches-tool-to-self-serve-quality-patches.md) 1.1.78 está instalado. A ID do patch é ACP2E-4665. Observe que esse problema está programado para ser corrigido no Adobe Commerce 2.4.9.

## Produtos e versões afetados

**O patch foi criado para a versão do Adobe Commerce:**

* Adobe Commerce (todos os métodos de implantação) 2.4.8-p3

**Compatível com as versões do Adobe Commerce:**

* Adobe Commerce (todos os métodos de implantação) 2.4.8 - 2.4.8-p4

>[!NOTE]
>
>O patch pode se tornar aplicável a outras versões com as novas versões do [!DNL Quality Patches Tool]. Para verificar se o patch é compatível com a sua versão do Adobe Commerce, atualize o pacote `magento/quality-patches` para a versão mais recente e verifique a compatibilidade na [[!DNL Quality Patches Tool]: página Procurar patches](https://experienceleague.adobe.com/tools/commerce-quality-patches/index.html?lang=pt-BR). Use a ID do patch como palavra-chave de pesquisa para localizar o patch.

## Problema

A REST API não retorna produtos secundários de um produto configurável quando um vídeo está presente na galeria de produtos.

<u>Etapas a serem reproduzidas</u>:

1. Adicione um vídeo ao produto secundário de um produto configurável.
1. Envie uma solicitação para o seguinte ponto de extremidade de API REST:
   `rest/v1/configurable-products/<PARENT_CONFIGURABLE_PRODUCT_SKU>/children`.

<u>Resultados esperados</u>:

A REST API retorna informações sobre o produto.

<u>Resultados reais</u>:

A API REST retorna o seguinte erro:

```
Error: Call to a member function getVideoProvider() on null in vendor/magento/module-product-video/Model/Product/Attribute/Media/ExternalVideoEntryConverter.php:87" exception.
```
